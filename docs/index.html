🌐 Nginx Reverse Proxy Ez

Easy & automated reverse proxy setup for Nginx.
No need to manually write config files. Just run the script and go!

🚀 Features

  • Direct or Cloudflare proxied support
  • Automatic domain A record verification
  • Auto-generate & enable Nginx config
  • Nginx syntax test & reload

💻 Usage

chmod +x setup-nginx-proxy.sh
./setup-nginx-proxy.sh

📦 Requirements

sudo apt install nginx dnsutils curl
